[FONT=Times New Roman]Shogunate Map alteration sub MOD alpha version Version 0.3.8 'Ashizuri' is a submod of the Shogunate mod. It replaces map with a high-resolution one and increases the Provence-Baron territory.[/FONT]

[COLOR=rgb(250, 197, 28)][SIZE=22px][I]CAUTION[/I][/SIZE][/COLOR]
[FONT=Times New Roman]This mod is an alpha version.
There are known problems and unimplemented features.
Probably when this mod is upgraded, the saved data will no longer be available.
Please refrain from playing games with the goal of long-term continuation.
It is not compatible with the main mod, Shogunate save data.
Since map size is larger than the CK3 vanilla and main mod, a more powerful hardware configuration will be required.[/FONT]

[COLOR=rgb(250, 197, 28)][SIZE=22px][I]Description[/I][/SIZE][/COLOR]
[COLOR=rgb(243, 121, 52)][SIZE=18px][I]Topography[/I][/SIZE][/COLOR]
[FONT=Times New Roman]It is based on a newly generated height map from a real DEM (Digital Elevation Model), and currently uses the orthographic projection method centered on Tokyo (this may change in the future).
Map textures are drawn using elevation and actual land cover information.[/FONT]

[COLOR=rgb(243, 121, 52)][SIZE=18px][I]Province[/I][/SIZE][/COLOR]
[FONT=Times New Roman]The number of provinces in a county (equivalent to a county in the game) was determined based on the standard of one provincia for every 10,000 KOKU[GOKU]（石） of rice produced in a county (equivalent to a county in the game) at the end of the 15th century, with some exceptions. The number of Provinces in a Count's Domain is subject to change.
This is the shape of Province, created by combining the administrative divisions of Japan as of 1920.
The basic data for the map, heightmap.png and provinces.png, are newly created at 16,384 x 4,096 pixels, twice the size of the main mod. The number of provinces is 3140 about four times the number of the main mod (including sea areas, etc.).
The increased number of Provinces will give you more strategic and tactical options and bring you a new gameplay experience![/FONT]

[COLOR=rgb(243, 121, 52)][SIZE=18px][I]County[/I][/SIZE][/COLOR]
[FONT=Times New Roman]Counties that existed in the 15th century have been reorganized to appear in the game, and the number of {z} county territories has been increased compared to the main mod.
Some counties were divided like counties that existed in modern times because they had a large number of Provinces or a large area.
Towns such as "Kyōto", "Ōsaka", "Nara", and "Sakai" were set up as independent county territories.[/FONT]

[COLOR=rgb(243, 121, 52)][SIZE=18px][I]Known Issues & Unimplemented Features[/I][/SIZE][/COLOR]
[LIST]
[*][FONT=Times New Roman]There are no rivers that can or cannot be traversed.[/FONT]
[*][FONT=Times New Roman]There are no special map objects such as mines.[/FONT]
[*][FONT=Times New Roman]Some Provinces exist in isolation.[/FONT]
[*][FONT=Times New Roman]Surround map does not exist.[/FONT]
[*][FONT=Times New Roman]Historically accurate names for barony title names and county title names have not yet been established.[/FONT]
[*][FONT=Times New Roman]Provins too small.[/FONT]
[*][FONT=Times New Roman]There is a provience that is not linked to the title.[/FONT]
[*][FONT=Times New Roman]The lake is dried up with no map objects in place.[/FONT]
[*][FONT=Times New Roman]There is a lake Provinces that is better off as an ocean than a non-navigable lake.[/FONT]
[*][FONT=Times New Roman]Baronial titles are duplicated before and after the name suffix.[/FONT]
[*][FONT=Times New Roman]No map locator, duplicate locator locations.[/FONT]
[*][FONT=Times New Roman]The newly added county historical data is just a copy of the connected existing historical data.[/FONT]
[*][FONT=Times New Roman]Barony title name history data in Shogunate MOD will not be reflected.[/FONT]
[*][FONT=Times New Roman]The ground texture displayed on the map is different from the historical one.[/FONT]
[*][FONT=Times New Roman]The winter function added in CK3 version 1.3 will not work.[/FONT]
[*][FONT=Times New Roman]Person's hairstyle. Clothing is from vanilla Step-Mongolian culture. Due to the author's unfamiliarity with 3D CG modeling, there are no plans to implement Japanese hairstyles or clothing.[/FONT]
